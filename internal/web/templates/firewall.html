{{template "icon-route"}} Add flowtable (offload)

A flowtable offloads established connections to the fast path — a big throughput win for a router. Bind it to the interfaces traffic forwards between, then reference it from a forward-chain rule with the flow add action.

Ingress-hook priority — a keyword (filter) or a number. Leave blank for the default.

{{template "icon-plus"}} New table

A table groups related chains. Most setups need just one, in the inet family — that one table covers both IPv4 and IPv6 at once. Pick a different family only if you know you need it.

inet = IPv4 + IPv6 together (recommended). ip = IPv4 only, ip6 = IPv6 only. arp/bridge/netdev are advanced.
Any short name (letters, digits, underscore), e.g. filter or nat.
